Rasmalai AI - Next Gen Conversational Suite
Rasmalai AI is a high-performance, professional-grade chatbot interface built with React and powered by the Google Gemini 1.5 Flash API. It features a clean, emerald-themed UI inspired by modern AI platforms like ChatGPT, ensuring a seamless user experience across both desktop and mobile devices.

✨ Key Features
⚡ Real-time Responses: Leverages Gemini 1.5 Flash for lightning-fast AI interactions.

📱 Fully Responsive: Mobile-first design with a collapsible sidebar and optimized touch targets.

🖼️ Multimodal Support: Capability to process and discuss images (vision-enabled).

🎨 Pro UI/UX: Clean, distraction-free interface with Glassmorphism effects and Tailwind-inspired aesthetics.

💬 Modern Chat Mechanics: Includes typing indicators, auto-scroll, and markdown-friendly message bubbles.

🔐 Secure Architecture: Environment variable support to keep API keys protected.

🛠️ Tech Stack
Frontend: React.js (Hooks, Context API)

Build Tool: Vite (for ultra-fast development)

Styling: Custom CSS3 & Modern Flexbox/Grid Layouts

Icons: Lucide React

AI Engine: Google Generative AI SDK

🚀 Getting Started
Follow these steps to get the project running locally:

1. Clone the repository
Bash
git clone https://github.com/YOUR_USERNAME/rasmalai-ai.git
cd rasmalai-ai
2. Install Dependencies
Bash
npm install
3. Environment Setup
Create a .env file in the root directory and add your Gemini API Key:

Code snippet
VITE_GEMINI_API_KEY=your_api_key_here
4. Run Development Server
Bash
npm run dev
📦 Deployment
This project is optimized for deployment on Netlify or Vercel.

Note for Netlify: 1. Connect your GitHub repo.
2. Set Build Command: npm run build
3. Set Publish Directory: dist
4. Add VITE_GEMINI_API_KEY in Environment Variables settings.
